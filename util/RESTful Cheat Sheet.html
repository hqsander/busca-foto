RESTful Routes

Tabela das 7 rotas do padrão RESTful

Nome Caminho Verbo HTTP Serve para Método no Mongoose
Index /trem GET Lista todos os trens Trem.find()
New /trem/new GET Exibe o form para um novo trem N/A
Create /trem POST Cria um trem, e depois redireciona para algum lugar Trem.create()
Show /trem/:id GET Exibe informações de um trem específico (id) Trem.findById()
Edit /trem/:id/edit GET Exibe o form para editar um trem preexistente Trem.findById()
Update /trem/:id PUT Atualiza/updata um trem, e depois redireciona para algum lugar Trem.findByIdAndUpdate()
Destroy /trem/:id DELETE Deleta um trem, e depois redireciona para algum lugar Trem.findByIdAndRemove()
